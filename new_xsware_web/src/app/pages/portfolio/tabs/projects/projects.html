Aplikacja Saldo Planer

Aplikacja webowa "Saldo Planer" została stworzona do planowania i analizy salda finansowego. Frontend został napisany w React i komunikuje się z warstwą REST API w Spring (Java), która z kolei korzysta z serwisu w Kotlinie do pracy z bazą MySQL.

@for (img of carousel2; track img.src) { }
Architektura rozwiązania
  1. Saldo Planer (React) – aplikacja SPA dostępna pod adresem app.xsware.pl.
  2. Frontend komunikuje się z XsWareApi – REST API napisanym w Spring/Java: Swagger XsWareApi.
  3. XsWareApi wywołuje serwis XsWareDbService napisany w Kotlinie, który odpowiada za komunikację z bazą danych MySQL: Swagger XsWareDbService.
  4. Całość działa na serwerze VPS (Linux), w kontenerach Docker, co ułatwia deploy, skalowanie i izolację środowisk.
